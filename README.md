# E-Commerce Produits

Application web full-stack de gestion de produits e-commerce, réalisée dans le cadre du cours **420-WA6-AG - Programmation Web Avancée** (Hiver 2026).

**Auteur :** Amari Samy

## Description

Cette application permet d'afficher une liste de produits, de les ajouter à un panier, et de gérer un système d'authentification (inscription/connexion). Les données sont stockées dans une base de données MySQL et exposées via une API REST développée avec Express.

## Technologies utilisées

- **Frontend :** React (Vite)
- **Backend :** Node.js / Express
- **Base de données :** MySQL
- **Authentification :** JWT (jsonwebtoken) + bcrypt
- **Tests :** Jest, Supertest, Cypress
